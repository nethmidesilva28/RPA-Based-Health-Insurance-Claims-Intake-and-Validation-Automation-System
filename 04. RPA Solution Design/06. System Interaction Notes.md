# System Interaction Notes

This section describes how the RPA bot interacts with external systems.

## 1. Email System
- Used to receive incoming claims
- Bot reads attachments and extracts data

## 2. Policy Management System
- Used to validate policy numbers
- Checks policy status and coverage details

## 3. Claims Management System
- Stores validated claim records
- Updated automatically by bot after processing

## 4. Document Storage System
- Stores supporting claim documents
- Bot uploads and links documents to claim records

## 5. Notification System
- Sends automated emails to claimants and internal teams
- Used for missing documents and status updates

## Integration Approach
- API-based integration preferred where available
- UI automation used where APIs are not available
- Secure authentication required for all system access

## Logging & Monitoring
- All system interactions are logged
- Logs used for auditing and troubleshooting
